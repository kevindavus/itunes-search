# _Itunes Searchitron 9000_

1- **Run the start command to start a hot reloading server.**

Npm: `npm start`

Yarn: `yarn start`

**The server will be up on the port 8080 by default `localhost:8080`**

2- **for production**

Npm: `npm run build`

Yarn: `yarn run build`
